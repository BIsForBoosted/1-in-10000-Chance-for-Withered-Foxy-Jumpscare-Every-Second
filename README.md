# 1/10000 Chance for Withered Foxy Jumpscare Every Second
A client side minecraft mod that has a 1/10000 change of triggering the FNaF2 Withered Foxy Jumpscare every second.

Use `/setjumpscareodds [X]` to set the jumpscare odds to 1/X
